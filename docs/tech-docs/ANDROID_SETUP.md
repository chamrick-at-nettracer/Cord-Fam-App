# Android Development Setup

Guide for setting up and developing the Android application.

## Status

**Current Status**: Not yet implemented

The Android app will be built using **Native Kotlin** with **Jetpack Compose**
for the UI.

## Planned Architecture

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Architecture**: MVVM (Model-View-ViewModel)
- **HTTP Client**: Retrofit + OkHttp
- **Local Storage**: Room Database or DataStore
- **API**: Same REST API as web frontend (`http://your-backend/api/v1`)

## Future Setup Instructions

When Android development begins, this guide will include:

### Prerequisites

- Android Studio (latest version)
- JDK 17+
- Android SDK
- Kotlin plugin

### Project Setup

1. Open Android Studio
2. Import project from `frontend/android/`
3. Sync Gradle
4. Configure API endpoint
5. Run on emulator or device

### Development

- Building the app
- Running tests
- Debugging
- Building APK for distribution
