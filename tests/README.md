# Tests

Shared test utilities and E2E tests for Cord-Fam-App.

## Structure

````text
tests/
├── e2e/              # End-to-end tests (Playwright)
├── fixtures/         # Test data fixtures
└── utils/            # Shared test utilities
```text

## Running Tests

### E2E Tests

```bash
npm run test:e2e
````

### All Tests

```bash
npm run test
```

See `docs/tech-docs/DEVELOPMENT.md` for more details.
